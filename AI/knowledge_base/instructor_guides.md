# Panduan Lengkap Instruktur (Instructor)

## Verifikasi Akun
1. Setelah register sebagai Instruktur, akses menu **Profil**.
2. Lengkapi data: Bidang Keahlian, Asal Lembaga, Lama Mengajar.
3. Upload **Bukti Kualifikasi** (Sertifikat/Ijazah) dalam format PDF atau Gambar, atau link portofolio.
4. Akun Anda akan ditandai "Terverifikasi" (simulasi: otomatis terverifikasi setelah upload).

## Manajemen Kursus

### Membuat Kursus Baru
1. Klik menu **Buat Kursus** (`/create-course`).
2. Isi formulir:
   - **Judul Kursus**: Nama kursus yang menarik.
   - **Deskripsi**: Penjelasan detail tentang apa yang dipelajari.
   - **Tipe Materi Utama**: Pilih Text, Video, atau Meeting.
   - **Thumbnail**: Upload gambar cover kursus.
   - **Premium**: Centang jika kursus berbayar.
   - **Harga**: Masukkan harga (jika Premium).
3. Klik **Simpan Kursus**.

### Mengedit & Menghapus Kursus
1. Buka **Dashboard Instruktur** (`/instructor-dashboard`).
2. Di kartu kursus, klik tombol **Edit** (ikon pensil) untuk mengubah info kursus.
3. Klik tombol **Hapus** (ikon sampah) untuk menghapus kursus permanen.

## Manajemen Materi (Lesson)

### Menambah Materi
1. Di Dashboard Instruktur, klik tombol **Kelola Materi** pada kursus yang diinginkan.
2. Klik **Tambah Materi Baru**.
3. Isi detail:
   - **Judul Materi**.
   - **Urutan**: Nomor urut materi.
   - **Konten**:
     - Jika Tipe **Text**: Tulis isi materi di editor.
     - Jika Tipe **Video**: Masukkan Link YouTube/Vimeo.
     - Jika Tipe **Meeting**: Masukkan Link Zoom/GMeet dan Waktu.
4. Klik **Simpan**.

## Manajemen Evaluasi

### Membuat Kuis
1. Di Dashboard Instruktur, klik tombol **Kelola Kuis**.
2. Klik **Tambah Pertanyaan**.
3. Masukkan pertanyaan dan 4 pilihan jawaban (A, B, C, D).
4. Pilih **Kunci Jawaban** yang benar.
5. Ulangi untuk menambah soal lain.
6. **Atur Jadwal Kuis**: Klik menu "Atur Tanggal Kuis" untuk menentukan kapan kuis bisa diakses (Start Date & End Date).

### Membuat Tugas (Exercise)
1. Di Dashboard Instruktur, klik tombol **Kelola Tugas**.
2. Isi Judul, Deskripsi Tugas, dan Link Soal (jika ada).
3. Tentukan **Deadline** (Tanggal Mulai & Selesai).
4. Klik Simpan.

## Manajemen Siswa

### Melihat Siswa Terdaftar
1. Di Dashboard Instruktur, klik tombol **Kelola Siswa**.
2. Anda akan melihat daftar siswa yang mengambil kursus ini.
3. Klik **Detail** pada siswa untuk melihat progress belajar mereka.

### Menilai Tugas
1. Buka menu **Kelola Siswa** -> **Detail Siswa**.
2. Di bagian Tugas/Exercise, lihat link tugas yang dikumpulkan siswa.
3. Berikan **Nilai** (0-100) dan **Feedback/Komentar**.
4. Klik **Update Nilai**.
